- ‚úÖ JWT tokens in httpOnly cookies
- ‚úÖ Rate limiting on code requests
- ‚úÖ Code expiry (10 minutes)
- ‚úÖ Max 3 verification attempts per code
- ‚úÖ Row Level Security (RLS) enabled

- ‚úÖ CSRF protection on auth endpoints
